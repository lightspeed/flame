# Flag

A set of small and readable flags.

<Preview>

### Props

<Props of={Flag} />

You can import a vast number of flags by simply changing the code or name prop.

**Do note that by using this component, you will be including every flag SVG within your app,
significantly increasing it's end bundle size.**

## Managing bundle size

Typically, flags should be imported directly:

```js
import React from 'react';
import { FlagCA } from '@lightspeed/flame/Flag/CA';
import { FlagCA_QC } from '@lightspeed/flame/Flag/CA_QC';

const MyComponent = () => (
  <div>
    <CA />
    <CA_QC />
  </div>
);

export default MyComponent;
```
